# yyyddyyd-timetables
如在哪个页面引入请在pages.json中将使用页面的"style"下的"navigationStyle": "custom"属性填写好

例如：
{
            "path" : "pages/xueyuan/Timetable/Timetable",
            "style" :                                                                                    
            {
                "navigationBarTitleText": "",
                "enablePullDownRefresh": false,
				"navigationStyle": "custom"
            }
            
        }
